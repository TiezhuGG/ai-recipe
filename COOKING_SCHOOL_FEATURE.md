# 去学厨房功能说明

## 产品概述

"去学厨房"是一个AI驱动的互动式烹饪教学平台，旨在帮助用户提升烹饪技能，解决烹饪过程中遇到的各种问题。

## 核心功能

### 1. AI对话模式 🤖

#### 功能描述
- 实时AI烹饪导师对话
- 随时解答烹饪相关问题
- 智能理解用户意图
- 提供专业且易懂的建议

#### 使用场景
- "如何让肉更嫩？"
- "炒菜怎么不粘锅？"
- "如何掌握火候？"
- "调味料的比例是多少？"

#### 技术实现
- 使用豆包AI的对话能力
- 专门的烹饪导师系统提示词
- 上下文理解和记忆
- 备用回答机制

#### 特色功能
- **快速提问**：预设常见问题，一键提问
- **对话历史**：保留完整对话记录
- **实时响应**：流畅的对话体验
- **友好界面**：清晰的消息气泡设计

### 2. 技巧知识库 📚

#### 功能描述
- 系统化的烹饪技巧学习
- 分类清晰的知识体系
- 循序渐进的课程设计

#### 知识分类
1. **刀工基础** 🔪
   - 切、片、丝、丁的标准技法
   - 8个课程

2. **火候掌握** 🔥
   - 大火、中火、小火的运用
   - 6个课程

3. **调味技巧** 🧂
   - 盐、糖、醋、酱油的黄金比例
   - 10个课程

4. **烹饪方法** 🥘
   - 炒、煎、炸、蒸、煮、炖
   - 12个课程

5. **食材处理** 🥩
   - 肉类、海鲜、蔬菜的预处理
   - 9个课程

6. **汤品制作** 🍜
   - 高汤、清汤、浓汤的秘诀
   - 7个课程

#### 学习路径
- 从基础到进阶
- 理论结合实践
- 视频+图文教学
- 互动练习和测试

### 3. 实时指导模式 🎯

#### 功能描述
- 分步骤烹饪指导
- 实时进度跟踪
- AI智能提示
- 语音/文字双模式

#### 使用流程
1. 选择要制作的菜谱
2. 开始实时指导模式
3. 按步骤完成每个操作
4. 获取AI实时提示
5. 完成整道菜的制作

#### 智能提示
- 每个步骤的关键点
- 常见错误预警
- 时间和火候提醒
- 安全注意事项

#### 特色功能
- **进度可视化**：清晰显示当前步骤
- **灵活导航**：可前进后退查看步骤
- **AI助手**：每步都有智能提示
- **暂停继续**：随时暂停，不影响进度

### 4. 问题诊断模式 🔍

#### 功能描述
- 上传菜品图片
- 描述遇到的问题
- AI智能诊断
- 提供解决方案

#### 诊断流程
1. 上传菜品照片（可选）
2. 详细描述问题
3. AI分析诊断
4. 获取解决方案和预防建议

#### 诊断内容
- **问题分析**：可能的原因
- **解决方案**：具体操作步骤
- **预防建议**：避免再次发生
- **相关知识**：扩展学习

#### 常见问题类型
- 菜太咸/太淡
- 肉炒不熟/炒老了
- 颜色不对
- 口感不好
- 粘锅问题
- 火候掌握

## 技术架构

### 前端实现

#### 组件结构
```
CookingSchoolView.vue
├── 导航栏 (NavigationBar)
├── 功能选择区
│   ├── 学习模式列表
│   └── 今日技巧
└── 主内容区
    ├── AI对话界面
    ├── 技巧知识库
    ├── 实时指导
    └── 问题诊断
```

#### 状态管理
- `selectedMode`: 当前选择的模式
- `chatHistory`: 对话历史记录
- `selectedRecipe`: 选中的菜谱
- `currentStep`: 当前步骤
- `diagnosisResult`: 诊断结果

### 后端实现

#### API端点
1. `POST /api/cooking/ask`
   - 询问烹饪问题
   - 参数：`{ question: string }`
   - 返回：`{ answer: string }`

2. `POST /api/cooking/diagnose`
   - 诊断烹饪问题
   - 参数：`{ problem: string }`
   - 返回：`{ diagnosis: string }`

#### AI服务方法
1. `answer_cooking_question(question: str)`
   - 调用豆包AI回答问题
   - 使用专门的烹饪导师提示词
   - 备用回答机制

2. `diagnose_cooking_problem(problem: str)`
   - 诊断烹饪问题
   - 分析原因和解决方案
   - 提供预防建议

## 使用场景

### 场景1：新手学习
**用户**：完全不会做饭的新手
**需求**：从零开始学习烹饪
**使用**：
1. 进入技巧知识库
2. 从刀工基础开始学习
3. 观看教学视频
4. 实践练习
5. 遇到问题随时问AI

### 场景2：遇到问题
**用户**：正在做菜的用户
**需求**：菜炒咸了不知道怎么办
**使用**：
1. 进入问题诊断模式
2. 描述"菜太咸了"
3. 获取AI诊断和解决方案
4. 按建议操作
5. 学习预防方法

### 场景3：学习新菜
**用户**：想学做新菜的用户
**需求**：第一次做某道菜
**使用**：
1. 在主页生成菜谱
2. 进入实时指导模式
3. 跟随步骤操作
4. 获取每步提示
5. 完成菜品制作

### 场景4：快速咨询
**用户**：有具体问题的用户
**需求**：快速获得答案
**使用**：
1. 进入AI对话模式
2. 点击快速提问或输入问题
3. 获取即时回答
4. 继续追问细节
5. 保存对话记录
